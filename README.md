# my GRC

サイトとキーワードを入力して検索順位をモニターする。

## 設計

```
+-------+     +----------+     +----------+
| sites |n---n| keywords |1---n| rankings |
+-------+     +----------+     +----------+
    1               1
    |      +--------|
    n      1
   +---------+
   | records |
   +---------+
```

* sitesは複数のキーワードをもつ
* keywordsは日付ごとに100位までのrankings情報を所持
* sitesとキーワードの関係を日付ごとにrecordsにて記録

Googleの順位だけを対象とし、iPhoneアクセスで100位までの情報を保存。
データはキーワードごとに毎日1回取得する。
